# Система классификации документов

Проект реализует автоматическую классификацию текстовых документов по тематикам с использованием методов машинного обучения.

## Описание

Система анализирует текст документа и определяет его категорию (например: финансы, IT, медицина, образование и др.).

В основе лежит алгоритм Naive Bayes classifier с использованием TF-IDF векторизации.
Также реализовано сравнение с альтернативными моделями (логистическая регрессия и SVM) с выбором лучшей.

Установить зависимости:

```
pip install -r requirements.txt
```

## Запуск

```
streamlit run app.py
```

После запуска откроется веб-интерфейс в браузере.

## Функционал

* классификация текстовых документов по категориям
* автоматическое обучение модели при запуске
* выбор наиболее точной модели
* веб-интерфейс для ввода текста

## Используемые технологии

* Python
* scikit-learn
* pandas
* nltk
* streamlit

## Примечания

* Для корректной работы требуется файл `dataset.csv` в папке `data/`
* При первом запуске автоматически загружаются необходимые ресурсы nltk
